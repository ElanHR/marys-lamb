# marys-lamb
Building a cute little bot that wins Noisebridgers hearts. And money.


# How to run code

Install the following (Python) dependencies (will make install file):

- opencv 
- numpy
- redis  
- Adafruit PCA9685 library 

## Running on pi

```
python main.py
```

## Running viewer on laptop

```
python ui.py
```
